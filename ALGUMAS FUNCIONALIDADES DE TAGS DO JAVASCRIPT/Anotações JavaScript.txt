Conceito de cliente/servidor:

Aparelho utilizado para acessar determinado serviço(celular, computador e etc) é o cliente.
Já o servidor é o hardware da empresa específica(wpp, youtube, netflix e etc) que está fornecendo os dados( vídeos, textos, arquivos e etc).
Esse processo ocorre quando você acessa um browser(google, firefox e etc) e digita a url(endereço) que vai te direcionar para o servidor e
o mesmo vai fornecer os dados necessários.
